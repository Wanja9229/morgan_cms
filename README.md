# Morgan Edition CMS

자캐(OC) 커뮤니티를 위한 올인원 CMS.
캐릭터 등록, 역극(RP), 상점, 전투, 세계관 위키, 의뢰 매칭 등 커뮤니티 운영에 필요한 시스템을 통합 제공합니다.
