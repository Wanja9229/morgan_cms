# Morgan Edition - 패치노트

> 공개 이후 변경사항을 기록합니다.

---

## 2026-03-19

### 새 기능
- **관계도 상세 모달** — 관계도에서 엣지(선) 클릭 시 양쪽 캐릭터 얼굴, 관계명 화살표, 메모를 모달로 표시
- **관계 목록 메모 표시** — 캐릭터 프로필 관계 목록에서 캐릭터명, 관계명, 메모를 세로 정렬 (20개 스킨 전체 적용)
- **관계도 양방향 화살표** — 관계도 엣지에 양방향 화살표 + 관계명 표시로 방향 인식 개선
- **관계 로그 설정 이동** — 관계 로그 필수 설정을 기본설정에서 관계 관리 페이지로 이동, 일괄 활성 전환 버튼 추가

### 개선
- 관리자 버튼이 대시보드로 연결되도록 변경

### 버그 수정
- 관리자가 자기 캐릭터끼리 보낸 관계 신청이 받은 신청 목록에 표시되지 않던 버그 수정
- 프로필 스킨 select에서 기본 스킨 옵션이 2개 표시되던 버그 수정
- 관계도 엣지 클릭이 동작하지 않던 버그 수정 (selectable 설정)
- 관계도 엣지 색상과 관계 목록 아이콘 색상 불일치 수정
- SPA 페이지 이동 시 스크롤이 최상단으로 초기화되지 않던 버그 수정
- 관계 모달에서 드래그 후 배경에서 마우스를 놓으면 닫히던 버그 수정

## 2026-03-18

### 새 기능
- **내 캐릭터 페이지 관계 탭** — 모든 캐릭터의 관계를 한 곳에서 관리 (받은 신청, 로그 대기, 보낸 신청, 활성 관계)
- **관계 로그 대상 선택** — 관계 로그 작성 시 어떤 관계에 대한 로그인지 선택 가능
- **관계 로그 필수 설정** — 관리자가 ON/OFF 전환 가능 (OFF 시 수락만으로 관계 성립)
- **로그 대기 일괄 전환** — 관리자 관계 관리에서 accepted 상태 관계를 일괄 active로 전환
- **캐릭터 전신 이미지** — 캐릭터 프로필 상단에 전신 이미지 표시 (등록된 경우)
- **세계관 위키 한줄요약** — 위키 목록에서 조회수 대신 한줄요약 표시

### 개선
- 캐릭터 신청 기간을 설정하지 않으면 승인된 캐릭터도 항상 수정 가능하도록 변경
- 관리자는 자기 캐릭터 프로필에서도 관계 신청 가능
- 마이페이지 캐릭터 관리 바로가기가 내 캐릭터 페이지로 연결
- 관리자 버튼이 대시보드로 연결되도록 변경

### 버그 수정
- 캐릭터 목록에서 탭/정렬 클릭 시 사이드바가 홈으로 포커싱되던 버그 수정
- 캐릭터 상세 페이지 진입 시 사이드바가 캐릭터 목록에 정상 포커싱되도록 수정
