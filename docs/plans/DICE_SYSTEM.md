# 댓글 주사위 시스템

## 개요

댓글 영역에 🎲 버튼을 두고, 누르면 0~100 랜덤 숫자가 댓글로 자동 등록.
서버에서 생성하므로 조작 불가. 역극 모집, 이벤트 당첨, TRPG 판정 등에 활용.

## 동작

1. 댓글 입력폼 옆 🎲 버튼 클릭
2. 서버에서 `rand(0, 100)` 실행
3. 댓글 자동 등록: `🎲 73`
4. 시스템 댓글 표시 (수정/삭제 불가, 별도 스타일)

## DB

`mg_comment` 기존 테이블에 추가:

| 컬럼 | 타입 | 설명 |
|------|------|------|
| cm_is_dice | tinyint DEFAULT 0 | 주사위 댓글 여부 |
| cm_dice_value | tinyint NULL | 주사위 결과값 |

## 옵션 (관리자 게시판별 설정)

| 설정 | 기본값 | 설명 |
|------|-------|------|
| 주사위 사용 | OFF | 게시판별 ON/OFF |
| 1인 1회 제한 | ON | 한 글에서 한 번만 |
| 범위 커스텀 | OFF | ON 시 글 작성자가 범위 지정 가능 (0~6 등) |

## 댓글 표시

```
┌──────────────────────────────┐
│ 🎲 73            다니엘  10:32 │
├──────────────────────────────┤
│ 🎲 91 ★          레이첼  10:33 │  ← 최고값 하이라이트
├──────────────────────────────┤
│ 60이상 첫번째 분이랑        소피아  10:30 │  ← 일반 댓글 (게시글 작성자)
│ 같이 로그 씁니다! 🎲 굴려주세요    │
└──────────────────────────────┘
```
