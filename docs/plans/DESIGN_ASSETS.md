## 우선순위 1 — 필수

### 1. 사이트 로고
- **사이즈:** 가로 160px 이내, 세로 ~32px (헤더용), 고해상도 원본 별도
- **현황:** 관리자에서 업로드 가능 (`mg_config('site_logo')`)
- **위치:** `theme/morgan/head.php` 헤더 좌상단
- **포맷:** PNG (투명 배경) 또는 SVG
- **비고:** 다크 배경에서 사용, 밝은 색상 권장

### 2. 파비콘
- **사이즈:** 16x16, 32x32 (ico), 180x180 (apple-touch-icon), 192x192 / 512x512 (PWA)
- **현황:** `theme/morgan/img/favicon.ico` 비어있음
- **위치:** `theme/morgan/head.sub.php`
- **포맷:** ICO + PNG 세트

### 3. 사이드바 아이콘 (8종)
- **사이즈:** 24x24px
- **현황:** Heroicons SVG 인라인 (generic 윤곽선)
- **위치:** `theme/morgan/head.php` (lines 147-222)
- **포맷:** SVG (fill) 또는 PNG @2x

|---|--------|------|------|
| 1 | 홈 | 집 윤곽 | 메인 페이지 |
| 2 | 게시판 | 말풍선 | 커뮤니티 (게시판 + 역극) |
| 3 | 내 캐릭터 | 사람 | 캐릭터 관리 |
| 4 | 캐릭터 목록 | 사람들 | 전체 캐릭터 열람 |
| 5 | 상점 | 쇼핑백 | 아이템 상점 |
| 6 | 새 글 | 종 | 최신 게시글 (보류 중) |
| 7 | 개척 | 건물 | 개척/시설 시스템 |
| 8 | 설정 | 톱니바퀴 | 사이트 설정 |

### 4. 캐릭터 기본 아바타
- **사이즈:** 256x256px (원형 크롭되어 사용)
- **현황:** SVG 사람 윤곽 플레이스홀더
- **사용처:** 캐릭터 이미지 미등록 시 전역 사용
  - `bbs/character_view.php`
  - `theme/morgan/skin/rp/list.skin.php` (역극 아바타)
  - `theme/morgan/skin/board/basic/list.skin.php` (게시판)
- **포맷:** PNG (투명 배경)
- **비고:** 세계관에 맞는 실루엣 캐릭터 느낌 권장

### 5. 상점 상품 기본 이미지
- **사이즈:** 200x200px (정사각형)
- **현황:** SVG 상자 윤곽 플레이스홀더
- **위치:** `theme/morgan/skin/shop/shop.skin.php`
- **포맷:** PNG (투명 배경)
